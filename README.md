# 书籍分析项目

## 项目文件简要介绍

- .gitee ( gitee 配置文件)
- BookAnalysis ( 项目代码 )
  - analysis ( 数据分析 )
  - data ( 数据 )
  - spiders ( 爬虫 )
  - static ( Web 静态文件 )
  - templates ( Jinja 模板 )
  - test ( 测试 )
  - utils ( 辅助函数 )
  - views ( 视图 )
  - \_\_init__.py ( 项目初始化 )
  - extensions.py ( 扩展 )
  - settings.py ( 设置 )
- Files ( 项目文档 )
- .gitignore ( git 忽略文件 )
- README.md ( 项目介绍 )
- wsgi.py ( 项目入口 )
